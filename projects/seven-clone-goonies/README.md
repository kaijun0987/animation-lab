# The Goonies Interactive Clone

This project is a visual and interaction-animation clone based on:

https://the-goonies.webflow.io/

It focuses on the loading progress sequence, cinematic hero and plot scroll animations, draggable expanded character cards, hover and video states for the Goonies cast grid, the "Goonies Never Say Die" coin sequence, animated credits tiles, skull transition, gallery reveal, share section, smooth scroll buttons, moving navigation indicator, and the bottom music/map controls.

## Development

```bash
pnpm dev
```

From the repository root, you can also run:

```bash
pnpm 7clone
```

## Build

```bash
pnpm build
```

From the repository root:

```bash
pnpm --filter @lab/seven-clone-goonies build
```
