# Pizza VS Burger Interactive Clone

This project is a visual and interaction-animation clone based on:

https://pizza-burger.webflow.io/

It focuses on the animated character that follows the pointer, responsive facial expressions, independently selectable pizza and burger voting buttons, rotating vote badges, hover reactions, and the layered outlined heading.

## Development

```bash
pnpm dev
```

From the repository root, you can also run:

```bash
pnpm 5clone
```

## Build

```bash
pnpm build
```

From the repository root:

```bash
pnpm --filter @lab/fith-clone-pizza-vs-burger build
```
