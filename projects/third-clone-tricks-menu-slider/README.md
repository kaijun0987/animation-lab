# T.RICKS Menu Slider Clone

This project is a visual and interaction clone based on:

https://tricks-menu-slider.webflow.io/#top

It focuses on the fullscreen curtain menu, staggered menu typography, curved image slider, and multi-state custom cursor interactions.

## Development

```bash
pnpm dev
```

From the repository root, you can also run:

```bash
pnpm 3clone
```

## Build

```bash
pnpm build
```
