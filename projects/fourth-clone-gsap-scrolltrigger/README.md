# MinimalGoods GSAP ScrollTrigger Clone

This project is a visual and scroll-animation clone based on:

https://minimal-goods.webflow.io/

It focuses on the large hero logo transition, sticky navigation color changes, masked headline movement, expanding sticky image circle, category pill layout, and scroll-driven product grid titles using GSAP ScrollTrigger.

## Development

```bash
pnpm dev
```

From the repository root, you can also run:

```bash
pnpm 4clone
```

## Build

```bash
pnpm build
```

From the repository root:

```bash
pnpm --filter @lab/fourth-clone-gsap-scrolltrigger build
```
