# Lithosquare Interactive Clone

This project is a responsive visual and interaction-animation clone based on:

https://lithosquare-preprod.webflow.io/

It focuses on reproducing the video Hero, responsive typography, smooth scrolling, animated navigation, sticky scroll-driven catchphrase, progressive text reveal, connected feature cards, pixel-grid transitions, and responsive mobile layouts.

## Tech Stack

- React
- TypeScript
- Vite
- GSAP and ScrollTrigger
- Lenis
- Tailwind CSS

## Requirements

Vite 7 requires Node.js `20.19+` or `22.12+`.

## Development

```bash
pnpm dev
```

From the repository root, you can also run:

```bash
pnpm 6clone
```

## Build

```bash
pnpm build
```

From the repository root:

```bash
pnpm --filter @lab/six-clone-lithosquare build
```

## Reference Sections

- `#anchor-about` - Sticky catchphrase, contact CTA, and feature cards
