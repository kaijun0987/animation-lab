# TimeFrame App Clone (Eight Clone)

This project is a detailed visual and animation clone based on the TimeFrame marketing website:

https://timeframe-app.webflow.io/

It features advanced scroll-driven animations, device mockup slides, and interactive layout transitions managed using GSAP, ScrollTrigger, and React.

## Development

To start the development server locally from the repository root:

```bash
pnpm 8clone
```

Or from inside this project folder:

```bash
pnpm dev
```

## Build

To compile and build the production bundle from the repository root:

```bash
pnpm --filter @lab/eight-clone-timeframe build
```

Or from inside this project folder:

```bash
pnpm build
```
